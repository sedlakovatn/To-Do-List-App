# To-Do List App [Responsive]
A sleek and interactive To-Do List App built to simplify task management and boost productivity. Perfect for organizing daily tasks and goals.

### 💻 Technologies
- HTML
- CSS
- Javascript

### 🚀 Features
- **CRUD Operations:** Manage tasks effortlessly by creating, editing, and deleting them whenever needed.
- **Local Storage:** Tasks are stored locally, ensuring they remain available after refreshing or reopening the app.
- **Progress Bar:** Easily track completed tasks and monitor your overall progress.
- **Confetti Effects:** Celebrate completed tasks with fun animations.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **Modern UI:** Clean, simple, and easy-to-use interface.

### 📍 The Process
I wanted to create a To-Do App that felt more modern and interactive than a simple checklist. Instead of focusing only on task management, I wanted the experience to feel motivating and visually engaging. I started with the basics—adding, editing, viewing, and deleting tasks—then added features like local storage to keep tasks saved, progress tracking for motivation, and confetti animations to celebrate completed tasks. I also focused on building a clean, responsive interface that works smoothly across different screen sizes while keeping the design simple and easy to use.

### ✨ UX & UI Improvements
This project was created as a learning exercise and inspired by a video tutorial from CSSnippets. The design and concept were adapted and extended with custom features, styling, and functionality for educational and portfolio purposes. My goal was to improve UX and UI of the app. Here is the list of the improvements:
- **Drag & Drop sorting:** After refreshing the page, the order reseted because task positions weren't saved. From a UX perspective, losing custom ordering feels like a bug, so I used sortable's onEnd callback.
- **Inline editing:** Previously, the task dissapeared completely, which could feel risky and confusing.
- **Adding Task Filters (all, active, completed):** Helps quickly find unfinished work, review completed items and makes the app scalable.
- **Button “Clear completed tasks”:** Previously, users must delete each one individually. Clear would significantly improve efficiency.
- **Replace icon-only buttons:** add hover for edit a delete.
- **Make the progress section more impressive:** Added motivational quotes "making progress" etc.


### 🚦 Running the Project
To view and interact with the TO DO LIST App:

1. Download the project files - index.html,script.js, and style.css.
2. Open the files in Visual Studio Code.
3. Use the Live Server extension in VS Code to view the project in your web browser.
